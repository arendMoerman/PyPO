Detailed Description

Enum types for units for setting scaling mode for quantities.

This special option is used for specifying the scaling mode of the heatmap for 2D plots, but also for the mode of beam cross-sections. In addition, the scaling mode argument sets the scale for fitting Gaussians. For example, using the mode argument, it is possible to fit a Gaussian to a beam pattern in linear, logarithmic and decibel space.

Options:
LIN : Set scaling mode to linear
dB : Set scaling to decibels
